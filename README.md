# Abhin M Raj - Portfolio

A modern, interactive portfolio website built with React, TypeScript, and Vite.

## Features
-   **Interactive Visuals**: Custom particle network background and interactive text animations.
-   **Supabase Integration**: Functional "Get in Touch" form backed by a real database.
-   **Responsive Design**: Fully optimized for all devices.

## Getting Started

### 1. Clone the repository
```bash
git clone <your-repo-url>
cd inkwell-portfolio
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Environment Setup
Create a `.env` file in the root directory and add your Supabase credentials:
```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 4. Run Locally
```bash
npm run dev
```

## Deployment
This project is ready for deployment on Vercel or Netlify.
**Important**: Remember to add your `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY` to your deployment platform's Environment Variables settings.
